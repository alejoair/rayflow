# Especialista: sistema `frontend-canvas`

The React Flow graph canvas: node rendering (NodeCard), the FlowCanvas itself with execution animations, and the flowDef-JSON <-> React Flow nodes/edges translator.

## Archivos (`rayflow_file_map.json` → `systems.frontend-canvas.files`)

| archivo | descripción |
|---|---|
| `rayflow/editor/frontend/src/components/FlowCanvas.tsx` | Visual flow graph editor using React Flow library, supporting node drag-drop, edge connections, live run visualization, and animation timing controls. |
| `rayflow/editor/frontend/src/components/NodeCard.tsx` | React Flow node component displaying node metadata, exec/data pins with type colors, runtime badges, and run status indicators. |
| `rayflow/editor/frontend/src/lib/rayflowHighlight.ts` | CodeMirror syntax highlighting and hover tooltips for Rayflow Python node decorators, pin declarations, and context methods. |
| `rayflow/editor/frontend/src/lib/translator.ts` | Bidirectional converters between React Flow graph format and Rayflow FlowDef JSON, with exec pin logic and type color mapping. |
