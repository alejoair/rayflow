# Especialista: sistema `frontend-panels`

Sidebar/footer editor panels: node palette, variables panel, custom-nodes CodeMirror editor, properties panel, runs panel, and the flow settings dialog.

## Archivos (`rayflow_file_map.json` → `systems.frontend-panels.files`)

| archivo | descripción |
|---|---|
| `rayflow/editor/frontend/src/components/CodeEditor.tsx` | Full-screen code editor for editing Python custom node source with CodeMirror, syntax highlighting, and save functionality. |
| `rayflow/editor/frontend/src/components/CustomNodesPanel.tsx` | Collapsible sidebar panel for managing custom Python node files with create, edit, and delete operations. |
| `rayflow/editor/frontend/src/components/FlowSettingsDialog.tsx` | Modal dialog for editing the flow's declared output pins (type selection, duplicate name validation). No input pins here — a flow's inputs live on its entry node (OnStart, ChatTrigger, a custom @entry_node, ...), not on the flow itself; that's a Python-level concern, not something this JSON-shaped dialog edits. |
| `rayflow/editor/frontend/src/components/NodePalette.tsx` | Categorized sidebar panel for dragging built-in and custom nodes onto the canvas with search filtering. |
| `rayflow/editor/frontend/src/components/PropertiesPanel.tsx` | Right sidebar showing selected node details, editable literal inputs, connected pins, validation errors, and output specs. |
| `rayflow/editor/frontend/src/components/RunsPanel.tsx` | Bottom panel for flow execution with input parameter entry, run history, and live output/error display during execution. |
| `rayflow/editor/frontend/src/components/VariablesPanel.tsx` | Left sidebar section for managing flow variables (name, type, default value) with inline editing. |
